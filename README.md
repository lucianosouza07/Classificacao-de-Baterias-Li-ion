## Pipeline de Data Science para Classificação de Baterias Li-ion

Este repositório contém um projeto de artigo científico - presente no diretório `/article/` - e implementação prática de Machine Learning focado na classificação e análise de baterias de íon-lítio (modelo 18650). O projeto segue uma estrutura rigorosa de pipeline de dados, abrangendo desde a extração (ETL) até a modelagem preditiva.

Dataset utilizado: https://ieee-dataport.org/open-access/complex-testing-procedure-18650-batteries

## Como Executar

### Pré-requisitos
- Python 3.8 ou superior
- Git (opcional, para clonar o repositório)

### Passos

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   cd <nome-do-repositorio>
   ```

2. **Instale as dependências:**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```

3. **Execute o notebook principal:**
   Abra e execute as células do arquivo `project.ipynb`. Os resultados (gráficos, métricas e relatórios) serão salvos nas pastas:
   - `images/`
   - `reports/`
 
# OBS: Se o diretório `/dataset` estiver vazio, o script gerará dados simulados.
