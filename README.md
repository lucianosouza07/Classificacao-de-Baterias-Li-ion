## Pipeline de Data Science para Classificação de Baterias Li-ion

Este repositório contém um projeto de artigo científico - presente no diretório "article" - e implementação prática de Machine Learning focado na classificação e análise de baterias de íon-lítio (modelo 18650). O projeto segue uma estrutura rigorosa de pipeline de dados, abrangendo desde a extração (ETL) até a modelagem preditiva.

# URL do dataset das baterias de íon-lítio (modelo 18650): https://ieee-dataport.org/open-access/complex-testing-procedure-18650-batteries

## Como Executar

1.  Certifique-se de ter o Python 3.8+ instalado.
2.  Instale as dependências:
    ```bash
    pip install pandas numpy matplotlib seaborn scikit-learn
    ```
3.  (Opcional) Coloque seus arquivos CSV/Excel na pasta `/content/dataset`. Se a pasta estiver vazia, o script gerará dados simulados.
4.  Execute o notebook `project.ipynb`. As imagens e relatórios serão salvos nos diretórios `images/` e `reports/`.
